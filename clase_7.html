1 EL FILTRO DE SEGURIDAD (GUARDRAIL)

¿Qué es un Guardrail? La IA no se manda sola, nosotros ponemos las reglas. Un Guardrail es un filtro (un if/else) que revisa lo que escribió el usuario ANTES de mandarlo a internet. Si el usuario escribe una palabra prohibida, el programa lo bloquea.
PASO A PASO:
  1. Abrí tu Colab TP1_Oreste_Abril de la clase pasada.
  2. Abajo de todo, agregá una celda nueva y pegá este código.
  3. Pegá tu API Key real en la línea 4.
  4. Elegí una palabra prohibida en la línea 12 (por ejemplo: "trampa", "copiarse", "violencia").
  5. Ejecutá el código e intentá hablar de ese tema prohibido. ¡Mirá cómo te bloquea!
import requests # 1. Configuración básica API_KEY = "gsk_PEGA_TU_CLAVE_ACA" URL = "https://api.groq.com/openai/v1/chat/completions" headers = {"Authorization": f"Bearer {API_KEY}", "Content-Type": "application/json"} # 2. Le pedimos el tema al usuario tema = input("¿Sobre qué querés que te hable la IA hoy? ") # 3. --- EL FILTRO DE SEGURIDAD --- palabra_prohibida = "trampa" # ← CAMBIÁ ESTA PALABRA POR LA QUE QUIERAS if palabra_prohibida in tema.lower(): print("🛑 ¡BLOQUEADO! No estoy autorizado a hablar sobre ese tema.") else: print("✅ Tema aceptado. Conectando con la IA...") # Si pasa el filtro, armamos la pregunta y la enviamos pregunta = f"Explicame brevemente sobre {tema}" payload = { "model": "llama-3.3-70b-versatile", "messages": [{"role": "user", "content": pregunta}] } try: response = requests.post(URL, headers=headers, json=payload) data = response.json() print("\n🤖 IA:", data["choices"][0]["message"]["content"]) except: print("⚠️ Falló la conexión.")
